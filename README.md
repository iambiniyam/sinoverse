## Overview

**Sinoverse** is a free, open-source Chinese learning platform that combines science-backed learning methods with comprehensive content. Master Mandarin with spaced repetition, native audio pronunciation, stroke-order animations, and 10,000+ HSK words.

### Why Sinoverse?

- **Science-Backed**: SM-2 spaced repetition algorithm for optimal retention
- **Native Audio**: 1,500+ pinyin syllable recordings from native speakers
- **Stroke Animations**: 20,000+ characters with stroke-by-stroke guides
- **Complete HSK 3.0**: All 9 levels with 10,000+ words
- **30,895 Idioms**: Comprehensive chengyu database with explanations
- **13,023 Sentences**: Real-world examples for contextual learning
- **129 Measure Words**: Essential classifiers with usage patterns
- **210+ Tech Terms**: IT & automotive software testing vocabulary
- **Comprehensive Dictionary**: Pinyin, meanings, examples, and audio
- **Interactive Quizzes**: Adaptive testing with instant feedback
- **PWA Ready**: Works offline, installable on mobile devices

## Getting Started

### Prerequisites

- **Node.js** 18+ (LTS recommended)
- **npm** 9+ or **pnpm** 8+
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

```bash
# Clone the repository
git clone https://github.com/iambiniyam/sinoverse.git
cd sinoverse

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173 in your browser
```

### Building for Production

```bash
# Create optimized production build
npm run build

# Preview production build
npm run preview
```

### Study Tips

1. **Daily Practice**: 20-30 minutes every day
2. **Review Consistently**: Don't skip due cards
3. **Use Audio**: Listen and repeat out loud
4. **Write Characters**: Practice stroke order
5. **Read Sentences**: Learn words in context
6. **Set Goals**: Track progress weekly
